# YOLOv11 на Raspberry Pi 5

Проверка работы модели детекции объектов **YOLOv11** на одноплатном компьютере
**Raspberry Pi 5** в режиме инференса на **CPU** (без AI-ускорителя и без камеры —
распознавание на готовых изображениях).

Цель проекта — убедиться, что современная модель компьютерного зрения запускается
на Raspberry Pi 5, и **оценить скорость её работы** на ARM-процессоре.

## Стек и железо

| Компонент      | Значение                                  |
|----------------|-------------------------------------------|
| Плата          | Raspberry Pi 5, 8 GB LPDDR4X              |
| Процессор      | Broadcom BCM2712, 4×Cortex-A76 (aarch64)  |
| ОС             | Raspberry Pi OS (64-bit, Debian)          |
| Модель         | YOLOv11n (nano)                           |
| Фреймворк      | Ultralytics 8.4.x, PyTorch (CPU-only)     |
| Режим          | Инференс на CPU                            |

## Структура проекта

```
yolov11-raspberry-pi/
├── README.md          — описание проекта (этот файл)
├── detect.py          — распознавание объектов на одной картинке
├── benchmark.py       — замер скорости инференса (среднее время, FPS)
├── requirements.txt   — зависимости
├── images/            — исходные картинки
│   └── input.jpg
└── results/           — результаты
    ├── input.jpg      — картинка с рамками
    └── benchmark.txt  — отчёт о замере скорости
```

## Установка (на Raspberry Pi)

```bash
# Системные пакеты
sudo apt update
sudo apt install -y python3-pip python3-venv libgl1

# Виртуальное окружение
python3 -m venv yolo
source yolo/bin/activate

# PyTorch ставим в CPU-версии (важно для Raspberry Pi — без CUDA-пакетов NVIDIA)
pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu

# Ultralytics (YOLOv11)
pip install ultralytics
```

> **Важно:** на Raspberry Pi PyTorch нужно ставить именно из CPU-репозитория.
> Обычная установка пытается подтянуть гигабайтные CUDA-пакеты NVIDIA, которые
> на Pi бесполезны (видеокарты NVIDIA нет) и обрывают установку.

## Запуск

Положить картинку в `images/input.jpg`, затем:

```bash
# Распознать объекты и сохранить результат с рамками в results/
python3 detect.py

# Замерить скорость инференса
python3 benchmark.py
```

## Выводы

- YOLOv11n **успешно запускается** на Raspberry Pi 5 на CPU.
- Скорость инференса — около **2.7 FPS** (≈371 мс на картинку), чего достаточно
  для обработки отдельных изображений и несложных офлайн-задач.
- Для реального времени (видео/камера) потребовался бы AI-ускоритель
  (Raspberry Pi AI HAT на базе Hailo) или более лёгкий формат модели (NCNN).

## Возможные улучшения

- Экспорт модели в формат **NCNN** — заметно ускоряет инференс на Pi.
- Сравнение моделей разного размера: `yolo11n` (nano) против `yolo11s` (small).
- Подключение AI HAT для ускорения на NPU.
